{:sum; ... :}
==================

Output the sum.

```html
{:sum; /literal/literal/ :}
```
Any delimiter can be used:

```html
{:sum; ~a~b~ :}
{:sum; #a#b# :}
{:sum; |a|b| :}
{:sum; XaXbX :}
...
```

Modifiers:
----------

```text
{:^sum; ... :}
```

### Modifier: ^ (upline)

Eliminates previous whitespaces, (See "unprintable" for examples.)
